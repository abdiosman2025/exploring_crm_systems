## Executive Summary
This project explores Customer Relationship Management (CRM) systems using Generative AI tools. It covers CRM concepts, product comparisons, open-source CRM evaluation, and system architecture design.

---

## AI Tools Used
- ChatGPT
- Microsoft Copilot

---

## CRM Research Findings
CRM systems are used by organizations to manage customer data, improve sales, marketing, and customer service. They help businesses track customer interactions and improve decision-making.

---

## CRM Product Comparisons
Commercial CRM systems:
- Salesforce
- HubSpot CRM
- Zoho CRM
- Microsoft Dynamics 365

Open-source CRM systems:
- SuiteCRM
- EspoCRM
- Odoo CRM

---

## Open Source CRM Evaluation
EspoCRM was used for evaluation. It provides contact management, lead tracking, dashboard views, and basic CRM functionality. It is simple and suitable for small to medium businesses.

Screenshots are included in the repository.

---

## CRM Architecture Proposal
A simple CRM system can be built using:
- HTML, CSS, JavaScript, Bootstrap (Frontend)
- PHP (Backend)
- MySQL (Database)

It includes modules like contacts, leads, tasks, and reporting.

---

## Lessons Learned
AI tools helped quickly understand CRM systems, compare products, and design architecture. However, some information required verification.
